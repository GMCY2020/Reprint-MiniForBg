## 2. 介绍

- 该项目为往期复刻系列的衍生, 可用于 [wallpaper 壁纸引擎](https://www.wallpaperengine.io/).
- 实时渲染米哈游角色动态图
- 角色目前包括 胡桃，纳西妲，芙宁娜，茜特菈莉
- 可自定义背景图片
- 可自定义背景颜色
- 项目所需动图等资源来自米哈游官网, 仅编程学习用
- 基本信息中的网站, 国内访问 GitHub 网络问题. 可能有延迟, 资源加载等问题. 项目下载到本地应该不卡.
- 个人学习自用

## 3. 环境

编辑器

- `VSCode`

框架

- `vue3`
- `spine`

## 4. 使用

- 下载 项目 并 解压
- 找到 `index.html` 文件
- 拖入 `WallPaper壁纸引擎` 即可使用.
